- Quiz forms were redirecting to `/recomendador-de-tarjetas-de-credito-p1` (404)
- Expected behavior: redirect to `/mx/recomendador-de-tarjetas-de-credito-p1` (200)
- Root cause: window.location.href bypassing Next.js basePath configuration

- `getLocalizedPath()`: Ensures paths always include /mx/ locale prefix

  - Idempotent: won't double-prefix paths already containing /mx/
- Updated `redirectWithUtmParams()`: Auto-detects internal URLs and adds locale prefix
